For windows users:
1. in terminal cd FreeDraw
2. then enter "npm i" then press enter
3. then start server using "npm run dev"

4. click + icon in vscode to make another terminal
5. then cd to Backend
6. write "python -m venv venv" then press enter
7. then  ".\venv\Scripts\activate" press enter
8. then "pip install -r requirements.txt"
